# 不规则孔洞使用说明

本版本在原来的 `sphere` / `ellipsoid` 孔洞基础上，新增了：

```yaml
hole_shape: "irregular"
```

它的含义是：

1. 孔洞中心和孔径仍然按原来的随机方法生成；
2. `min_pore_diameter` 和 `max_pore_diameter` 仍然控制孔洞的等效尺寸；
3. 删除单元时，不再使用严格球形或椭球形边界；
4. 孔洞边界会加入随机起伏；
5. 最终孔壁由有限元网格单元边界自然形成，所以看起来是不规则孔。

## 推荐配置

```yaml
hole_shape: "irregular"
min_pore_diameter: 0.5
max_pore_diameter: 1.0
irregularity_strength: 0.25
irregularity_modes: 4
```

## 参数含义

### irregularity_strength

控制孔洞边界不规则程度。

| 数值 | 效果 |
|---|---|
| 0 | 接近球孔 |
| 0.10 | 轻微不规则 |
| 0.20~0.35 | 推荐范围 |
| >0.50 | 孔边界很粗糙，可能产生细碎单元 |

### irregularity_modes

控制边界起伏复杂度。

| 数值 | 效果 |
|---|---|
| 1~2 | 起伏简单 |
| 3~6 | 推荐范围 |
| >8 | 边界更复杂，但计算略慢 |

## 注意

不规则孔洞不是严格 CAD 曲面孔，而是基于“删除有限元单元”形成的孔洞。

因此：

- 孔洞边界会跟随网格；
- 网格越细，孔形越接近设定的随机边界；
- 网格越粗，孔洞越呈现块状或锯齿状；
- 实际孔径是近似值，不是精确几何直径。

这正好适合当前项目的路线：

```text
完整实体网格
→ 随机生成孔洞参数
→ 判断单元中心是否落入孔洞
→ 删除这些单元
→ 得到多孔有限元模型
```
